# 1、网络地址和物理地址

网络地址： 

互联网协议地址（IP地址）为互联网上每一个网络或主机分配一


个逻辑地址，IP地址工作在网络层


IP的分类：IPV4   IPV6






物理地址： 

物理地址（MAC地址）为每一个设备设置一个固定的硬件地址，


MAC地址工作在链路层


MAC地址：00-23-5A-15-99-42

# 2、TCP/IP 五层常见协议

应用层协议： FTP、HTTP、SMTP、Telnet、DNS等


传输层协议： TCP、UDP等


网络层协议： IP、ICMP、ARP等


数据链路层协议： PPP协议等


物理层： 不常用
