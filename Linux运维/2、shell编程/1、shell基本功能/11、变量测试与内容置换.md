# **x=${y-new}  :y有值包括空值，x=y;否则x=new;**

![](images/WEBRESOURCE04ab5b1303a2956f3c92fb23b6703393截图.png)

## **例子：**

```javascript
[root@bogon test]# echo ${z-new}
new
[root@bogon test]# z=1
[root@bogon test]# echo ${z-new}
1
[root@bogon test]# z=""
[root@bogon test]# echo ${z-new}

[root@bogon test]# 
```
