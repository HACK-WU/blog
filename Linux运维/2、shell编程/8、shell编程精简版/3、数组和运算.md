# **1、数组**

## 1）语法格式

```
语法格式：array_name=(value1 ... valuen)
示例：
my_array=(A B C D)
array_name[0]=value0
array_name[1]=value1
array_name[2]=value2
```

## 2）读取数组

```
读取数组：${array_name[index]}
获取数组中的所有元素：
my_array[0]=A
my_array[1]=B
my_array[2]=C
my_array[3]=D
echo "数组的元素为: ${my_array[*]}"
echo "数组的元素为: ${my_array[@]}"
```

## 3）获取数组的长度

```
获取数组的长度：
my_array[0]=A
my_array[1]=B
my_array[2]=C
my_array[3]=D
echo "数组元素个数为: ${#my_array[*]}"
echo "数组元素个数为: ${#my_array[@]}"
```

# 2、算数运算

## 1）运算符

```
+ - * / % ** ...
增强赋值：
+=，-=，*=，/=，%=
乘法符号有些场景中需要转义 : *\
bash有内建随机数生成器：$RANDOM
```

## 2)完成算数运算

```
(1) let var（变量名）=算术表达式
(2) var=$[算术表达式]
(3) var=$((算术表达式))
(4) var=$(expr argl arg2 arg3 …)
```
