**查看域名解析的工具**

# 1、nslookup

windows上在 cmd 中直接输入使用

```
C:\Users\Aaron>nslookup iproute.cn
服务器: public1.114dns.com
Address: 114.114.114.114
非权威应答:
名称:  cqfz53.coding-pages.com
Address: 124.156.193.111
Aliases: iproute.cn
```
