# 1、数字求和（1到100）

- 简单的实现

就是一个一个加呗

```python
end_num = 103
total = 0
for num in range(1, end_num + 1):
    total += num
print(total)
```

- 利用规律，1一个数加上最后一个数，然后做乘法

```python
end_num = 103
suffix = 0
if (end_num % 2 != 0):
    suffix = end_num
    end_num -= 1
    
res = (1 + end_num) * end_num // 2 + suffix
print(res)
```

- 使用递归算法

```python
def foo(n, res):
    res += n
    if n == 0:
        return res
    return foo(n - 1, res)
res = foo(100, 0)
print(res)
```
