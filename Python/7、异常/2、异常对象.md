# 1、except

- 如果except后不跟任何内容，则此时它会捕捉所有的异常

- 如果在except后跟着一个异常的类型，那么此时它就只会捕捉该类型的异常

- exept的数量不受限制，一个try中可以出现多个except

- Exception是所有异常的父类

- 可以在异常类型的后边跟着一个as xxx，此时xxx就被赋予了异常信息

```javascript
try：
    pass
except NameError:
    print("出错了")
except ZeroDivisionError as e:
    print(e)
except:
    print("未知异常")
```

# 2、finally

- 无论是否出现异常，该子句中的代码块都会执行

```javascript
try：
    pass 
except:
    pass
else:
    pass
finally:
    print("必须执行的语句")
```
