# 1、raise

- raise用于向外部抛出异常，后边可以跟一个异常类，或异常类的实例

```javascript
def add(a,b):
    if a<0 or b<0:
        raise Exception("两个参数中不能有负数")
    r=a+b
    return r
print(add(123,456))
```

# 2、自定义异常

- 也可以自定义异常，只需要创建一个类继承Exception

```javascript
class MyError(Exception):
    pass
```
