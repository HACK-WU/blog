# 1、读取模式t和b

- t:读取文本文件（默认值）

- b:二进制文件

- 读取文本文件时，size是以字符为单位

- 读取二进制文件时，size是以字节为单位

```javascript
file_name="wyp.jpg"
with open(file_name,mode="rb") as file_obj:
    #将读取到的内容写出来
    new_name="new.jpg"
    with open(new_name,mode="wb") as new_obj:
        #定义每次读取的大小
        chunk=1024*100  #每次读取100K
        while True:
            content=file_obj.read(chunk)
            if not content:
                break
            new_obj.write(content)

```
