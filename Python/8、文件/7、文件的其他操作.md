import os

# 1、os.listdir(),获取指定目录结构

- 需要一个路径作为参数，会获取到该路劲下的目录结构，默认路径为".",当前目录

- 该方法会返回一个列表，目录中的每一个文件（文件夹）的名字都是这个列表中的元素
