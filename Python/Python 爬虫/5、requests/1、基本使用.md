# 3、response的属性以及类型

```
.response的属性以及类型
类型             ：models.Response    
    r.text     : 获取网站源码           
    r.encoding ：访问或定制编码方式           
    r.url      ：获取请求的url           
    r.content  ：响应的字节类型           
    r.status_code    ：响应的状态码    
    r.headers        ：响应的头信息 
```
