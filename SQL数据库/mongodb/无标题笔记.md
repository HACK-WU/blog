# 配置文件

```shell
systemLog:
   destination: file
   path: "/mongodb/single/log/mongod.log"
   logAppend: true

storage:
   dbPath: "/mongodb/single/data/db"
   journal:
      enabled: true

processManagement:
   # 开启后台启动上
   fork: true

net:
   # 允许访问mongodb服务的ip，如果允许任何ip访问，配置当前linux的私有IP地址。如果只允许本地访问，配置localhost
   bindIp: 192.168.23.10     
   port: 27017
```
