# 2、本地模式、伪分布式、完全分布式区分依据

主要的区别依据是NameNode、DataNode、ResourceManager、NodeManager等模块运行在几个JVM进程、几个机器。如下表所示：

| 模式名称 | 各个模块占用JVM进程数 | 各个模块运行在几台机器上 | 
| -- | -- | -- |
| 单机 | 1 | 1 | 
| 伪分布式 | N | 1 | 
| 完全分布式 | N | N  | 
| HA+完全分布式 | N | N |
